Features

Feature Description
Sync All Bookmarks Sync all your bookmarks with Instapaper, Pocket, Readability, Pinboard and Delicious
RSS Feeds Sync all your feeds with Feedly, Fever, NewsBlur, Feed Wrangler, Feedbin or use ReadKit's built-in RSS engine
Customisable Styles Control font style, face, alignment, height, line spacing and page width
Read Offline The browser is capable to save links for offline viewing
Toggle Views Switch between original and stripped down content with a click
Open in Browser Preference to open links either in Feedlr or in your default browser
Clutter-Free Reading Choose from pre-formatted simple views so you can focus on the content
Bookmarks as Favorites Add, remove, move, archive and mark bookmarks as favourites
Focus Mode Focus mode: reading without interface elements
